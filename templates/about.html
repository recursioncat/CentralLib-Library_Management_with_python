How was This Website Made

This website was made as a full stack development proof-of-concept using HTML and CSS for frontend, Javascript for animations and Functionalities and Flask which is a python based Backend library.

The Core functionality depends on two very basic classes made with vanilla python, one is a "BOOK" class containing the individual details of each book, and other is a "Library" class for storing and organizing those books. Each book has properties associated with it like the path to the pdf file, last read page, path to the thumbnail, etc. Additionally each book has an unique ID. All the book object along with their details are stored in a Library, which reads from a Pandas Dataframe stored in the Server with the pickle module.

When you click on a book, the server page requests the image of the last opened page using that Book's ID, which is handled by python using the PyPDF2 module and sent back to the page and rendered along with the reader UI.
